# devtools
A collection of gulp and nodejs utilities used everyday.

#### Installation:

  $ git clone https[git]://github.com/thatjs/devtools.git

  $ cd devtools<br />
  $ npm install -g gulp  (mac: prefix with sudo)<br />
  $ npm install<br />
  $ npm update

#### Prerequisties:

Nodejs 0.10.xx

XCode 6.4+ (mac)
